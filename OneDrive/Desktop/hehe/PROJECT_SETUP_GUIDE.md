# Project Setup Guide — Postiz Competitor MVP

**Time Required:** ~30 minutes  
**Prerequisites:** Node.js 18+, PostgreSQL 14+, Docker (optional)  
**Launch Date:** June 1, 2026

---

## Step 1: Initialize Backend (NestJS + Prisma)

```bash
# Navigate to project root
cd "C:\Users\TUMMA\OneDrive\Desktop\hehe"

# Create backend directory
mkdir -p backend
cd backend

# Initialize NestJS project
npm init -y
npm install @nestjs/core @nestjs/common @nestjs/platform-express @nestjs/jwt @nestjs/passport passport passport-jwt typeorm class-validator class-transformer @prisma/client prisma bcrypt dotenv cors

# Install dev dependencies
npm install -D @nestjs/cli @types/express @types/node @types/bcrypt typescript ts-node @typescript-eslint/eslint-plugin @typescript-eslint/parser eslint prettier

# Create NestJS structure
npx @nestjs/cli@latest new . --strict --skip-git

# Initialize Prisma
npx prisma init

# Create .env (PostgreSQL)
cat > .env << 'ENVEOF'
DATABASE_URL="postgresql://user:password@localhost:5432/postiz_mvp"
JWT_SECRET="your-secret-key-change-in-prod"
NODE_ENV="development"
PORT=3001
ENVEOF

# Generate Prisma migrations for initial schema
npx prisma migrate dev --name init
```

## Step 2: Initialize Frontend (Next.js + shadcn/ui)

```bash
# From project root
cd "C:\Users\TUMMA\OneDrive\Desktop\hehe"

# Create frontend directory
mkdir -p frontend
cd frontend

# Initialize Next.js 14
npm create next-app@latest . --typescript --tailwind --eslint --app

# Install shadcn/ui + dependencies
npx shadcn-ui@latest init -d

# Install additional packages
npm install zustand axios framer-motion recharts

# Create .env.local
cat > .env.local << 'ENVEOF'
NEXT_PUBLIC_API_URL="http://localhost:3001"
ENVEOF

# Test frontend build
npm run build
```

---

## Step 3: Set Up PostgreSQL Database

**Option A: Docker (Recommended)**
```bash
# From project root
docker run --name postiz-db \
  -e POSTGRES_USER=user \
  -e POSTGRES_PASSWORD=password \
  -e POSTGRES_DB=postiz_mvp \
  -p 5432:5432 \
  -d postgres:14
```

**Option B: Local PostgreSQL**
```bash
# Create database manually
createdb postiz_mvp
psql postiz_mvp
# Run Prisma migrations from backend/ directory
```
